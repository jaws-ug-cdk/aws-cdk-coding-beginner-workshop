# Contributing Guidelines

このプロジェクトへの貢献に興味を持っていただきありがとうございます。
バグ報告・新機能・修正・ドキュメント追加など、コミュニティからのフィードバックを歓迎します。

## バグ報告 / 機能要望

GitHub の Issue トラッカーをご利用ください。報告の際は、<!-- TODO: このワークショップのリポジトリの Issue URL に置き換える -->[既存の Issue](https://github.com/jaws-ug-cdk/aws-cdk-coding-beginner-workshop/issues)
に同じ内容がないかご確認ください。再現手順・環境・期待する挙動などを添えていただけると助かります。

## Pull Request

1. リポジトリを fork する
2. `main` の最新を取り込んだ状態で作業する
3. 変更は目的を絞る（大きなフォーマット変更は混ぜない）
4. ローカルでビルド・プレビューが通ることを確認する（`cd workshop && hugo`）
5. 明確なコミットメッセージで PR を作成する
6. CI の結果を確認し、レビューに対応する
